A Tree-Map Viewer in Swing

John D. Ramsdell
MITRE Corporation

This library contains a Swing implementation of Tree-Map visualization as presented in Ben Shneiderman, "Tree Visualization with Tree-Maps: A 2-D Space-filling Approach", ACM Transactions on Graphics (TOG), Vol. 11, No. 1, Pages 92-99, Jan., 1992.

To use the library to view disk usage, execute the jar file with: 

$ java -jar dutmv.jar

The jar is built from the sources using ant. This software is distrubuted under the terms of the GNU Lesser General Public License.
